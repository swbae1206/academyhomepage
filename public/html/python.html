Python/Django 과정

Python/Django 과정은 비교적 배우기 쉬운 Python 언어로 개발된 웹 프레임워크이자 풀 스택(Full Stack) 프레임워크인 Django 프레임워크를 학습합니다.

Python 언어는 인터프리터 방식의 프로그래밍 언어로 실 생황에서 사용하는 영어와 비슷해서 읽고 쓰기 쉬운 특유의 문법과 그로 인해 전 세계의 개발자들로부터 만들어진 수많은 패키지로 상당히 인기있는 프로그래밍 언어입니다. 실생활에서는 엑셀 자동화, 파일 처리 자동화, 웹 크롤링 자동화, 3D 모델링 자동화와 같이 업무 시간을 단축해 주는 언어로 활용되고 있고, 특히 AI/Big Data 분야에서 머신러닝구현하기 위해 사용되는 구글이 개발한 텐서플로우(Tensorflow)를 사용하는데 있어 가장 많이 사용되는 언어입니다.

Dyango 웹 프레임워크는 Python 기반 웹 프레임워크 중에 가장 널리 사용되는 풀 스택 웹 프레임워크입니다. 기본적으로 MVT(Model, View, Template) 패턴 형태의 구현 방식을 따르며, 잘 정의되어 있는 ORM(Object Relational Mapping) 기능과 기본적으로 제공되는 관리자 기능, 사용자 인증 등 고도의 데이터베이스 기반 웹 사이트를 작성하는데 아주 강력한 프레임워크 입니다. 대표적 사이트로 인스타그램이 있습니다.


주요 강의 내용
▷ Python 언어
  • 기본 Data 유형 맟 활용 & 제어 흐름 도구 학습
  • 함수 정의 및 매개변수 사용, 람다 표현식, 함수 어노테이션 학습
  • 자료구조(리스트, 튜플, 집합, 딕셔너리 등)와 루프 테크닉 학습
  • 모듈 및 패키지 생성 및 활용 기술
  • 입력과 출력, json 구조 및 활용, 에러와 예외 처리 기술 학습
  • 클래스의 정의 및 활용, 표준 라이브러리 활용 기술 학습
▷ Django 웹 프레임워크
  • 모델과 데이트베이스 정의 (Query, Raw SQL Queries, 검색) 및 관리자 기능 학습
  • URL Dispatcher, View 작성하기, Generic View, Middleware 등) HTTP 요청 처리 기술 학습
  • Django Form, Form 클래스를 이용한 Form 생성 및 활용 기술 학습
  • 템플릿 생성 및 관리 & Django 템플릿 언어(템플릿 Tags) 학습
  • 클래스 기반 View 및 Built-in 클래스 기반의 제너릭 View 활용 기술 학습
  • 사용자 계정, 그룹, 권한 및 쿠키기반의 Django에서의 사용자 증명 학습
