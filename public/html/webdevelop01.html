웹 개발 기초 과정

웹 개발 기초 과정은 HTML, CSS, JavaScript 및 Bootstrap을 이용하여 클라이언트 측 웹 페이지 상의 표시 계층을 설계하고 구현하는 기본 기술을 습득하는 매우 중요한 과정입니다. 이 과정을 이수하면 반응형 웹 사이트 및 웹 앱을 구현할 수 있습니다. 하지만, 대부분의 경우 이 과정만으로는 웹 개발 전문가가 되기 위해서는 충분하지 않습니다. 이 과정을 마치고 웹 개발 전문가 과정을 추가로 이수해야 비로소 웹 애플리케이션을 직접 개발 할 수 있는 웹 개발 기술을 확보할 수 있습니다.

주요 강의 내용

▷ HTML

웹 페이지를 만들기 위한 표준 마크업 언어

  • 기본 요소(Elements)
  • 기본 속성(Attributes)
  • 레이아웃(Layout)
  • 테이블(Table)
  • 폼(Forms)
  • 그래픽스(Graphics)
  • 미디어(Media)
  • API(Drag&Drop, Web Storage 등)
▷ CSS

HTML로 작성된 문서의 표시 방법을 기술하기 위한 스타일 시트 언어

  • CSS 기초(Selecters)
  • 속성(Attributes)
  • 이미지(Images)
  • 조합기(Combinators)
  • 의사 클래스(Pseudo-classes)
  • CSS 고급(Media Queries, Transforms, Animations)
  • 반응형 CSS(Responsive CSS)
  • SASS 기초
▷ JavaScript 기초

웹 페이지의 각종 구성요소들을 프로그램에 의해 동적으로 처리하기 위한 스크립트 언어

  • Statements, Data Types, Operators 등
  • 객체(Objects)
  • 함수(Functions)
  • 다큐먼트 객체 관리(DOM)
  • 브라우저 객체 관리(Browser BOM)
  • Web API(Forms, History, Storage, Workers, Fetch 등)
  • JSON(Syntax, Data Types, Objects, Array, JSON Server 등)
  • jQuery 기초
▷ Bootstrap5

모바일 우선의 반응형 웹 사이트 구현을 위한 프레임뭐크

  • 컨테이너(Container)
  • 타이포그래피(Typography))
  • 기본 요소(Tables, Buttons, Navs, Dorpdowns 등)
  • 유틸리티(Utilities)
  • 플렉스(Flex)
  • Bootstrap5 폼(Forms)
  • Bootstrap5 그리드(Grid)
  • 기본 템플릿(Templates)

* 각 강의 과목마다 Small Project 포함.

* JavaScript 고급과정은 React/Node.js 과정에 포함됨.


강의 기간 및 고려사항

▶ 필수 선행 학습 코스 : 없음
