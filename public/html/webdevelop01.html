웹 개발 기초 과정

HTML/CSS/JavaScript/Bootstrap
웹 개발을 위한 기초 과정으로,
본 과정을 이수하면 웹 사이트 및 웹 앱 개발이 가능합니다

웹 개발 기초 과정은 HTML, CSS, JavaScript 및 Bootstrap을 이용하여 클라이언트 측 웹 페이지 상의 표시 계층을 설계하고 구현하는 기술을 습득하는 과정입니다.
이 과정을 이수하면 반응형 웹 사이트 및 웹 앱을 구현할 수 있습니다. 이 과정을 마치고 웹 개발 전문가 과정을 이수하면 웹 응용 애플리케이션을 개발 할 수 있는 웹 개발 전문 기술을 확보할 수 있습니다.

주요 강의 내용

▷ HTML

웹 페이지를 만들기 위한 표준 마크업 언어

  • 기본 요소(Elements)
  • 기본 속성(Attributes)
  • 레이아웃(Layout)
  • 테이블(Table)
  • 폼(Forms)
  • 그래픽스(Graphics)
  • 미디어(Media)
  • API(Drag&Drop, Web Storage 등)
  • Small Project 실습
▷ CSS

HTML로 작성된 문서의 표시 방법을 기술하기 위한 스타일 시트 언어

  • CSS 기초(Selecters)
  • 속성(Attributes)
  • 이미지(Images)
  • 조합기(Combinators)
  • 의사 클래스(Pseudo-classes)
  • CSS 고급(Media Queries, Transforms, Animations)
  • 반응형 CSS(Responsive CSS)
  • SASS 기초
  • Small Project 실습
▷ JavaScript 기초

웹 페이지의 각종 구성요소들을 프로그램에 의해 동적으로 처리하기 위한 스크립트 언어

  • Statements, Data Types, Operators 등
  • 객체(Objects)
  • 함수(Functions)
  • 다큐먼트 객체 관리(DOM)
  • 브라우저 객체 관리(Browser BOM)
  • Web API(Forms, History, Storage, Workers, Fetch 등)
  • JSON(Syntax, Data Types, Objects, Array, JSON Server 등)
  • jQuery 기초
  • Small Project 실습
▷ Bootstrap5

모바일 우선의 반응형 웹 사이트 구현을 위한 프레임뭐크

  • 컨테이너(Container)
  • 타이포그래피(Typography))
  • 기본 요소(Tables, Buttons, Navs, Dorpdowns 등)
  • 유틸리티(Utilities)
  • 플렉스(Flex)
  • Bootstrap5 폼(Forms)
  • Bootstrap5 그리드(Grid)
  • 기본 템플릿(Templates)
  • Small Project 실습

* 각 강의 과목별 Small Project를 포함하고 있습니다.

개인정보처리 방침
원IT아카데미의 개인정보처리방침은 다음과 같습니다.
가. 수집 개인정보 항목 및 방법
나. 개인정보 수집 및 이용 목적
다. 수집한 개인정보의 보관 및 이용기간
라. 동의를 거부할 권리 고지와 동의 거부에 따른 불이익 사항

가. 수집 개인정보 항목 및 방법
원IT아카데미는 고객의 온라인상담(온라인상담신청, 수강료조회, 시간표조회, Newsletter 수신)을 위해 개인정보를 아래와 같이 수집하고 있습니다.
- [필수]이름, 핸드폰, 이메일
원IT아카데미는 다음과 같은 방법으로 개인정보를 수집합니다.
- 홈페이지 내 온라인상담신청, 수강료조회, 시간표조회, Newsletter 수신를 통해 개인정보를 수집하고 있습니다.
나.개인정보 수집 및 이용 목적
원IT아카데미는 수집한 개인정보를 다음의 목적을 위해 활용합니다.
- 과정문의에 대한 담당자의 전화 및 이메일 상담
다.수집한 개인정보의 보관 및 이용기간
모든 검토가 완료된 후 5년간 이용자의 조회를 위하여 보관하며, 이 후 해당정보를 지체없이 파기합니다.
라. 동의를 거부할 권리 고지와 동의 거부에 따른 불이익 사항
이용자는 원IT아카데미 홈페이지에서 수집하는 개인정보에 대해 동의를 거부할 권리가 있으며, 동의 거부 시에는 회원가입 및 인터넷 수강료 조회, 온라인 상담 등의 서비스가 일부 제한 됩니다.
