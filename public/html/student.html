  고등학생(대입 수시 SW 인재전형)

고등학생의 경우 아래와 같이 대입 수시 SW 인재 전형을 위한 과정과 일반적인 IT 기술 선행 학습 과정으로 구분합니다. 고등학생의 강의 내용은 일반인의 강의 코스와 거의 비슷합니다.

위 두가지 경우 모두 학년 및 코딩 등의 경험에 따라 학습내용이 달라질 수 있으므로, 수강 전 충분한 상담이 반드시 필요합니다.

대입 수시 SW 인재 전형 과정

대입 수시 SW 인재 전형을 위한 과정은 준전문가 수준의 IT 기술 학습 과정 외에 학생부 관리를 함께 진행하며, 그에 맞추어 프로젝트 실습과 포트폴리오/면접 특강 등에 비중을 두고 강의를 진행합니다.

주요 강의 구성

▷ 학생부 관리
  • 학년별/학기별 IT 관련 학생부 활동 설계
  • 학생부 활동에 맞는 IT 기술 학습 및 결과물 생성
  • 실적물과 포트폴리오에 연계
▷ IT 기술 학습
  • 웹 개발 기초과정
  • 웹 개발 고급과정(React/Node.js, Python/Django 과정 중 선택)
  • AI/빅테이터 과정
▷ 프로젝트 실습과정
  • 대입 수시 SW 인재 전형을 위한 학습 성취 결과물 생성
    (학습 성취 결과물은 학년과 남은 입시 기간에 따라 다름)
▷ 포트폴리오/면접 특강
  • 학습 성취 결과물의 포트폴리오 생성(문서화)
  • 자기 소개서 작성 코칭
  • 대입 수시 대비 면접 준비

IT 기술 선행 학습 과정

IT 기술 학습 과정은 웹 개발 기술 및 AI/빅데이터 등 IT 기술 선행학습에 중점을 두고 강의하며, 준전문가 수준의 IT 기술을 확보할 수 있습니다.

주요 강의 구성

▷ IT 기술 선행 학습
  • 웹 개발 기초과정
  • 웹 개발 고급과정(React/Node.js, Python/Django 과정 중 선택)
  • AI/빅테이터 과정
▷ 프로젝트 실습과정
  • 학습 성취 결과물 생성
    (선택과정)

향후 대입 수시 SW 인재 전형을 위한 사전 준비과정으로서도 최적의 학습과정이라 할 수 있습니다.
또한, 기술의 습득 정도에 따라, 일부 고급과정을 학습할 수도 있습니다.

주요 강의 구성

▷ 웹 개발 기초과정
  • HTML, CSS, JavaScript, Bootstrap 기초 기술
  • 웹 사이트 및 웹 앱 개발 기술 습득
▷ 프로그래밍 언어 및 AI/빅데이터 개념 학습
  • Python 언어 학습
  • AI/빅데이터 기본 원리 및 개념 학습
▷ 웹 개발 고급과정 선행학습
  • 웹 개발 고급과정에 대한 선행 학습
    (일정 수준의 웹 개발 기초 기술을 확보한 경우에 한함)
