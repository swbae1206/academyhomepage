React/Node.js 과정

React는 프론트엔드(Front-End) 개발자 사이에서 가장 인기있는 Facebook에서 만든 SPA(Single Page Application) 개발 기술입니다. 리렌더링이 잦은 최근의 복잡하고 동적인 웹 어플리케이션을 개발함에 있어 생산성이 아주 높은 라이브러리(Library)이기 때문에 프론트엔드 개발 기술에서 상당한 빠른 속도로 대세로 떠올랐습니다.
Node.js는 JavaScript로 브라우저 밖에서 서버를 구축하는 등의 코드를 실행할 수 있게 해주는 런타임 환경입니다. JavaScript라는 프론트엔드 필수 언어로 Node.js로 백엔드(Back-End)까지 작성할 수 있다는 엄청난 장점 때문에 출시 이후로 빠르게 점유율을 높여가고 있습니다.
React/Node.js 개발 기술은 최근 기업체에서 신규로 개발하는 웹 애플리케이션이나 빠른 결과물을 만들어내고자 하는 스타트업 등에서 가장 인기있는 웹 개발 기술로, 많은 IT 기업체에서 React/Node.js 기술을 보유하고 있는 기술인력을 필요로 하고 있습니다.


주요 강의 내용
▷ React

Virtual DOM을 활용하여 웹 페이지의 DOM 요소를 동적으로 처리
컴포넌트(Component) 기반의 모듈형 개발의 모던 웹 개발 기술
React Hooks, API 연동, React 라우터 구현 기술
컨텍스트 API(Context API), 리덕스(Redux), 리덕스 미들웨어(Redux-thunk, Redux-saga) 등 구현기술

▷ Sass

CSS의 단점을 보완하기 위해 만든 CSS 전처리기
CSS의 단순 반복되는 부분등 불편함을 해소시켜주는 스타일시트 언어
Sass로 작성된 React 컴포넌트를 이해하고 활용하는 기술

▷ Node.js

JavaScript로 서버를 구축하는 등의 코드를 실행할 수 있게 해주는 런타임 환경
논블로킹(Non-blocking) I/O와 단일 스레드 이벤트 루프를 통한 높은 처리 성능 기술
내장 HTTP 서버 라이브러리를 이용한 웹 서버의 동작 구현 기술
웹 및 모바일 애플리케이션을 위한 강력한 기능과 간결하고 유연한 Node.js 웹 애플리케이션 프레임워크인 Express 구현 기술
